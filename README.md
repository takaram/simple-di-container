# simple-di-container
これはPHPerKaigi 2026でのトーク『DIコンテナとAutowireの「魔法」を実装して理解する』で使用する、最小のDIコンテナのサンプルコードです。

> [!WARNING]
> このコードはDIコンテナの学習・理解のためのものです。
> エラーハンドリングなどを省略しており、実際のプロジェクトでの使用には堪えません。

## 手元で動かしてみる
1. `git clone`する
2. `composer install`する
   - 依存ライブラリはありませんが、オートローダーを生成するために必要です。
3. `php examples/index.php`を実行する
   - トーク内では3段階に分けてデモをします。各段階のコードを見て/動かしてみたい場合は、`demo/step{1,2,3}`の各タグにcheckoutしてください。

## 動作確認環境
PHP 8.5.3, 8.2.30で動作確認しています。
おそらくPHP 8.0以降であれば動作すると思いますが、未確認です。
